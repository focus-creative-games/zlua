# zlua

zlua是一个高效、稳定、优雅的针为il2cpp和.net **极致优化** 的现代Unity lua脚本方案。

## 为什么选择 zlua

Unity平台已经有革命性的原生C#热更新方案[HybridCLR](https://github.com/focus-creative-games/hybridclr)，也有很成熟的xlua、tolua
之类的lua方案，为何我们还需要一个新的lua解决方案？有以下原因：

- 现有的lua方案都存在与il2cpp运行时交互低效的问题，通过针对il2cpp的**极致**优化可以数倍提升交互性能
- 现有的lua方案都存在与il2cpp运行时交互产生大量GC的问题，虽然一般有提供针对值类型和的优化方案，但有较多限制，通过针对il2cpp的优化可以无感优化掉绝大多数这类GC问题以及string之类其他的GC问题
- 现有的方案没有对工作流充分优化，我们需要一个接入和维护更方便，功能更丰富的方案
- 现有的lua方案要么已经数年不维护，要么维护频率极低，基本丧失积极维护的动力，需要一个更敏捷的紧跟Unity和团结引擎版本变化的方案

zlua目标是提供一个**极致高效**、稳定、敏捷、富有创新的充分满足商业游戏项目需求的现代化的Unity lua脚本方案。
